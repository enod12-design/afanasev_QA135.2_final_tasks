Jenkins

  • pipeline - Основной блок всего пайплайна.
  • agent - Определение места запуска (напр., agent any или agent { docker { ... } }).
  • stages - Набор этапов сборки.
  • stage - Отдельный этап (например, stage('Build'), stage('Test')).
  • steps - Список команд внутри этапа (например, sh 'mvn clean install').
