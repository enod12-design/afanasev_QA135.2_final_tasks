Git

Настройка

  • git config --global user.name "Имя" — настройка имени.
  • git config --global user.email "email" — настройка почты.

Создание и клонирование

  • git init — инициализировать новый локальный репозиторий.
  • git clone — клонировать существующий репозиторий.

Работа с изменениями

  • git status — проверить состояние файлов (изменены, добавлены).
  • git add <файл> — добавить конкретный файл в индекс.
  • git add . — добавить все измененные файлы в индекс.
  • git commit -m "сообщение" — зафиксировать изменения с комментарием.
  • git diff — просмотреть отличия, которые еще не добавлены в индекс.

Удаленные репозитории (GitHub/GitLab)

  • git remote add origin — привязать удаленный репозиторий.
  • git push -u origin <ветка> — отправить изменения на удаленный сервер.
  • git pull — получить и слить изменения с удаленного сервера.
