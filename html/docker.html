Docker

  • docker container create — создает контейнер из выбранного образа.
  • docker container start — активирует уже существующий контейнер.
  • docker container run — создает новый контейнер и сразу включает его.
  • docker container ls — отображает все существующие контейнеры.
  • docker container rm — удаляет выбранный контейнер (он должен быть выключен, чтобы команда сработала).
